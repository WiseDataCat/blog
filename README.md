## Postgres

Remember to install `psycopg-binary` so that `pgcli` could work correctly.
For other tasks in Python, use `psycopg2-binary`.

```
podman run -d \
    --name postgres-local \
    -e POSTGRES_DB=mydb \
    -e POSTGRES_USER=myuser \
    -e POSTGRES_PASSWORD=mypassword \
    -p 5432:5432 \
    -v pg-data:/var/lib/postgresql/data \
    postgres:16

pgcli postgresql://myuser:mypassword@localhost:5432/mydb
```
